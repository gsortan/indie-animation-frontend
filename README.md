# Indie Animation Hub — Frontend

The frontend for **Indie Animation Hub**, a web platform for discovering and exploring independent animated series and shorts from the internet.

The application provides a searchable catalogue of animations with advanced filtering, responsive design, and detailed animation pages that connects to an Express server backend. 

Demo Link: https://indie-animation-frontend-production.vercel.app/

---

## Features

### Animation Catalogue
- Browse a catalogue of independent animations
- Responsively designed to adapt to mobile, tablet and desktop 
- Pagination support

### Search and Filtering
Users can filter animations by:

- Tags
- Rating range
- Episode count range
- Release year range
- Creator name
- Synopsis keywords

### Animation Detail Pages
Each animation includes:

- Poster artwork
- Title
- Ratings and rating count
- Detailed animation information

- React
- TypeScript
- Vite
- TailwindCSS
- Material UI

## Installation

```
Clone the repository:git clone https://github.com/gsortan/indie-animation-frontend.git

Install dependencies:npm install

Run development server: npm run dev
```
## Build

```
npm run build
 ```
